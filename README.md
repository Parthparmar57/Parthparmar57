## 🚀 Featured Projects

### 🔁 SkillSwap — Peer-to-Peer Skill Exchange Platform
> *"What if students could teach each other and learn anything for free?"*

A full-stack **MERN** platform where students list their skills, browse others', and **request skill swaps** — no money required, just knowledge.

| Feature | Detail |
|--------|--------|
| 🔐 Auth | JWT-based user authentication |
| 🤝 Matching | Skill-match algorithm for user pairing |
| 💬 Requests | Send/receive/accept skill swap requests |
| 📱 UI | Responsive React + Tailwind interface |

**Stack:** `MongoDB` · `Express.js` · `React` · `Node.js` · `JWT` · `TailwindCSS`
